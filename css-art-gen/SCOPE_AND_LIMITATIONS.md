**We cannot create photorealistic portraits.**

CSS has fundamental limits:
- No fine textures (skin pores, wrinkles)
- No photo-quality hair
- No realistic eyes
- Limited color blending

**Why?** CSS uses geometric shapes (divs, borders, shadows). It's inherently abstract.

**What we can do:** Stylized, impressionistic, artistic portraits (think oil painting, not photo).

### 2. Complex Scenes
